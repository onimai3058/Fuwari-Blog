## 服务器的选择

首先，你需要选择你喜欢的服务器平台并购买或领取喜欢的服务器产品

推荐：

[雨云游戏服]：

免费MCSManager面板服

注：签到获得积分，服务器用积分兑换、积分续费

[雨云链接](https://www.rainyun.com/ikun3058_)

[简幻欢]：

公益免费、低延迟、高配置、极速部署、数据安全、灵活变配

[简幻欢](https://simpfun.cn/)

不论是什么MC服务器平台，基本都有面板服（面板服不需要过多操作，如果你想要自己搭建，[请点击查看其他文章](/posts/003-mc高级玩家开服教程)）

我的服务器是雨云的，优点自然是MC服免费，而且IP啥的都很稳定，老云服务器厂了

## 配置服务器

面板服不需要过多操作，直接进行配置即可

点击“重装/更新游戏”

![P1](/img/002/P1.png)

选择你的游戏、游戏版本

![P2](/img/002/P2.png)

点击执行操作后，进入MCSM面板

![P3](/img/002/P3.png)

修改服务器启动指令

这是默认的启动指令

```
java -Xms128M -XX:MaxRAMPercentage=85.0 -jar Paper-1.21.11.jar
```
你可以对他进行自定义操作

Xms128MB，128MB为内存分配

-jar Paper-1.21.11.jar，其中Paper-1.21.11.jar为你的服务核心名称

（雨云后台执行重装后会自动修改启动指令）

进入服务端配置文件

![P4](/img/002/P4.png)

（如果没有这两个文件则表示你一次也没有启动实例,请启动实例后重新进入查看）

在server.properties文件中可以修改非常多关于服务器的配置

其中就包括在线（正版）验证、游戏模式，OP玩家数量，等等

（如果你通过离线登录无法进入服务器，请修改“在线（正版）验证”）

JAVA服务器server.properties配置说明（来自雨云）：[https://www.rainyun.com/docs/guide/Minecraft/Minecraft_Guidebook](https://www.rainyun.com/docs/guide/Minecraft/Minecraft_Guidebook)

配置完成后，在MCSM面板点击启动，开启服务端

前往雨云服务器界面查看IP（或者雨云的地址）以及端口

再去客户端对应版本进入游戏即可畅玩

（本文比较省略了，可以看看雨云B站号的一些视频）

注意，如果关闭了正版验证，你的服务器可能遭受攻击，袭击者可能通过MOTD扫描到服务器IP，获取OP（服务器管理员）的UUID，然后以这位OP玩家的身份执行指令，进行炸服操作，如果你既不想要购买正版，还要进入服务器，还要不被炸服，[请点击查看外置登录教程](/posts/004-mc服务器外置登录)

## 绑定域名，通过域名进入游戏

如果你不想通过IP（或者是雨云的地址）+端口进入你的MC服务器，你就可以绑定一个域名

首先购买一个域名，推荐[SpaceShip](https://www.spaceship.com/zh/)，这是海外的一个域名服务商，可以使用支付宝支付，并且域名超级便宜,然后通过[CloudFlare](https://dash.cloudflare.com/)平台（赛博大善人）绑定上域名

特别注意，请尽量选好域名，并不建议经常更换服务器域名

准备好域名了之后，找到DNS记录（DNS就是一个指路人，帮助客户端找到IP地址）然后根据下文内容正确填写DNS记录

在雨云后台可以看到，有写SRV记录的具体填写方式

![P5](/img/002/P5.png)

这是我服务器的填写，当作一个例子，照着我的填写就好了

![P6](/img/002/P6.png)

名称部分，雨云给的是“__minecraft._tcp”如果按照这个填写就是直接使用根域名（没有前缀）

加入你购买的域名是adb.com。那么你就直接通过adb.com进入服务器

如果你想要在你的域名前面添加上一个前缀，如你想要通过mc.adb.com进入服务器

那么你在名称部分填写“__minecraft._tcp.mc”其余保持一样就可以

## 进入游戏开始游玩

启动MC，点击多人游戏

添加服务器

输入你的游戏地址（在雨云游戏云控制面板显示）

点击进入以加入游戏

## 配置MOD

如果你在配置服务器的时候，选择了Fabric或者forge端，那么你就可以安装mod

![P2](/img/002/P2.png)

打开MCSM控制面板，进入文件管理，打开“mods”文件夹

![P7](/img/002/P7.png)

上传你所需要的mod

注意:MC的MOD分服务端和客户端，一般情况下来说，在MC百科上就可以看到是否需要服务端

你也可以直接给你客户端中的所有MOD全部上传到服务器上，避免了一个一个去看MOD性质

如果出现兼容性问题请查看时候安装了前置MOD

若你的MOD出现问题请查看MC百科[https://www.mcmod.cn/](https://www.mcmod.cn/)

最后预祝各位成功
